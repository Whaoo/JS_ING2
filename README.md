# How to ? : 

##### use gulp : 
```
node --version
npm --version
npx --version

npm install --global gulp-cli
npx mkdirp my-project
cd my-project
npm init
npm install --save-dev gulp
gulp --version
```
Launch the gulpfile with the command : ```gulp```
